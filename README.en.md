# MorseGz

Compress the text in Gzip format and output it in Morse code.

## Usage

### 1. Encoding

```bash
$ python3 encode.py "Btw, I use Arch."
Gzipped hex:
1F8B0800E7CA826902FF732A29D751F054282D4E55702C4ACED00300A0C5454C10000000
Morse:
.____/.._./___../_.../_____/___../_____/_____/./__.../_._./._/___../..___/_..../____./_____/..___/.._./.._./__.../...__/..___/._/..___/____./_../__.../...../.____/.._./_____/...../...._/..___/___../..___/_../...._/./...../...../__.../_____/..___/_._./...._/._/_._././_../_____/_____/...__/_____/_____/._/_____/_._./...../...._/...../...._/_._./.____/_____/_____/_____/_____/_____/_____/_____
```

### 2. decoding

Decoding Morse Code  

```bash
$ python3 decode.py .____/.._./___../_.../_____/___../_____/_____/./__.../_._./._/___../..___/_..../____./_____/..___/.._./.._./__.../...__/..___/._/..___/____./_../__.../...../.____/.._./_____/...../...._/..___/___../..___/_../...._/./...../...../__.../_____/..___/_._./...._/._/_._././_../_____/_____/...__/_____/_____/._/_____/_._./...../...._/...../...._/_._./.____/_____/_____/_____/_____/_____/_____/_____
Btw, I use Arch.
```

Decoding from hexadecimal  

```bash
$ python3 decode.py -h 1F8B0800E7CA826902FF732A29D751F054282D4E55702C4ACED00300A0C5454C10000000
Btw, I use Arch.
```
